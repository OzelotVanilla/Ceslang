Cesno Asm

Simplified Operator

For example,
mov ax, bx == ax = bx
add ax, bx == ax + bx
mov ax, 0; mov bx, 0; mov cx, 0; mov dx, 0 == rst x
mov bx, 0000; mov ax, [bx] == mov sc, [0]

Safe pop

pop ax; mov [si-2], 0; mov [si-1], 0 == pops ax

Example

If using original Assembly.

mov ax, 2
mov ax, bx

If using CesAsm.

#type cesasm

ax = 2
ax = bx
pshs ax
